Layout your directory in the following way:
  peanut \
   install.peanut
   uninstall.peanut
   key.peanut
   package-info.peanut
   
  install.peanut should contain an installation script written in Bash
  uninstall.peanut should contain an uninstallation script written in Bash
  key.peanut should contain a key to sign the package with, generated by the package manager
  Once you are all done, install peanut and enter in your terminal:
  peanut --upload https://github.com/github-username/repo-name/peanut
  example: peanut --upload https://github.com/peanutbuttermurmite.com
  OR 
  enter:
  peanut --upload directory-to-peanut
  example: peanut --upload /home/peanut
